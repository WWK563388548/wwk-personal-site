### How to run this repo
1. Clone the project
2. Run command to install dependencies
```
npm install
# or
pnpm i
```
3. Modify the data from `/lib/data.tsx`, if you want use your own information
4. Run command to run server locally
```
npm run dev
# or
pnpm run dev
```
